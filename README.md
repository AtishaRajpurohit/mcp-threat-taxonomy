MCP Threat Taxonomy — Interactive Matrix
An interactive visualization of the security threats facing Model Context Protocol (MCP) servers, mapped across two axes: the four phases of the MCP server lifecycle (Creation, Deployment, Operation, Maintenance) and the type of adversary behind each threat (Malicious Developer, External Attacker, Malicious User, and latent Security Flaws). The matrix plots all 16 threat scenarios by their point of origin — the lifecycle stage where each risk is introduced — rather than where it ultimately fires, since several threats (such as tool poisoning and rug pulls) are planted during Creation but only triggered at runtime. Toggle between Risks and Mitigations to switch each threat from its description and point of occurrence to its recommended defenses, and select any threat to see the lifecycle phases it spans. The taxonomy and lifecycle model are drawn directly from Hou et al.'s survey of the MCP ecosystem.
